# CRUD NODE

Crud with Node.

## Getting Started

Clone project

```
git clone https://github.com/Andreisft/crud-nodeJs.git
```

* Install dependecies

```
npm install
```

* Start application

```
npm start
```

Navigate to `http://localhost:3000`

## Built With

* [Node](https://nodejs.org/en/)
* [TypeScript](https://www.typescriptlang.org/)
* [Express](https://expressjs.com/pt-br/)
